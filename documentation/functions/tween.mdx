### Function

```lua
function module.tween(...)
    local data = {...}
    local tweens = table.create(#data)

    for i, entry in ipairs(data) do
        tweens[i] = module.createTween(entry[1], entry[2], entry[3])
        tweens[i]:Play()
    end
end
```

### Example

```lua
k.tween(
    {workspace.Baseplate, k.tweenInfo(0.5, "Circular", "Out"),
        {
            Position = Vector3.new(0, 64, 0),
            Color = Color3.fromRGB(255, 0, 0)
        }
    }
)
```

## Advantages

* Can take multiple parts and properties at the same time.
* Other utilities like `tweenInfo()` work with the `tween()` function to provide consistency.
* Services like `TweenService` aren't needed.
