---
title: Installation
description: "How to install the KUtils module"
---

Installing the KUtils module is a short and easy process.

<Steps>
  <Step title="Install the module">
    You can find the latest version of KUtils on the marketplace.
    Additionally, you can find and download older versions of KUtils in the Updates tab.
  </Step>
  <Step title="Put it in your game">
    Once you've installed it, insert it in `ReplicatedStorage`.
    We recommend putting it in a modules folder.
  </Step>
  <Step title="Use it">
    Now, you can use all of KUtils' functions in any scripts!
  </Step>
</Steps>
