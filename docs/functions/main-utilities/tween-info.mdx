### Function

```lua
function module.tweenInfo(
  duration: number,
  style: string,
  direction: string
)
  return TweenInfo.new(
    duration,
    Enum.EasingStyle[style],
    Enum.EasingDirection[direction]
  )
end
```

### Example

```lua
local info = k.tweenInfo(0.5, "Sine", "Out")
```

### Advantages

- Extremely short, simple, and readable
- Doesn't require using `TweenInfo.new()`
